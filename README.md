# SEO Manager

## Introduction

> SEO Manager Generates a SEO Report Easily

* Keyword list
* Open Graph, Twitter Card, and oEmbed Metadata
* HTML JS CSS Analyse
* PageSpeed Insights
* W3C (HTML / CSS)
* Code Quality
* Duplicate content
* Indexation
* Bing - Webmaster Tools
* Google Search Console
* And more !

## Docs

For read documentation :
```bash
npm run generate-docs
```

## Installation

Download and install all dependencies
```bash
npm install
```

Run
```bash
npm run start
```

Build
```bash
npm run build
```
